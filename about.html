About Campus Life Planner

Campus Life Planner is a web application that helps departments and administrative teams keep track of campus events, classes, workshops, and other activities in one place.The system allows users to create, edit, delete, search, and sort activities while keeping track of important schedules throughout the academic term. Activities can be categorized as either campus events or classes, making it easier to manage different types of records.


To support planning and decision-making, the application also provides a dashboard with activity statistics, upcoming events, and currently ongoing activities. Data is stored locally in the browser and can be imported from or exported to JSON files for backup and sharing purposes.


Additional features include search with regular expressions, duration display settings, theme customization, accessibility support, and responsive layouts for different screen sizes.
